# おんめも / Onmemo

> Cross-device note sharing web app.

さまざまな端末でメモを共有する。

[🔗 Live Site](https://on-memo.kk-web.link/)

## ✨ Features

- 📝 Instant note sharing across devices
- 🔍 Algolia-powered search
- 📱 Responsive layout

## 🛠 Tech Stack

- Next.js + React + TypeScript
- Algolia (search)

## 🚀 Development

```bash
npm install
npm run dev
```
